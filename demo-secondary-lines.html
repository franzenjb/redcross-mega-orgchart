üìã How It Works

  • Solid lines represent direct hierarchical reporting relationships (e.g., Regional CDOs report to their Regional Executives)
  • Dotted lines represent secondary functional reporting relationships (e.g., all Regional CDOs also report to the Division CDO)
  • This dual reporting structure is common in matrix organizations where employees have both hierarchical and functional reporting lines
  • The visualization maintains clarity by using different line styles while keeping both types of connections visible
